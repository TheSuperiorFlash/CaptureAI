Privacy Policy for CaptureAI

Effective Date: November 26, 2025

Overview

CaptureAI captures screenshots and sends them directly to OpenAI's API using your personal API key. We do not collect, store, or have access to your data.

Data Collection

What We Store Locally (On Your Device Only)

  • OpenAI API Key: Stored securely in Chrome's local storage for API authentication
  • Extension Settings: Auto-Solve status, Ask mode status, UI visibility preferences
  • Last Capture Area: Coordinates for Quick Capture feature

What We Do NOT Collect

  • Screenshots or captured content
  • Usage analytics or telemetry
  • Personal information
  • Browsing history

How Your Data is Used

Screenshots: Sent directly from your browser to OpenAI's servers. We never receive or store this data.

API Key: Used only to authenticate requests to OpenAI. Stored locally and never transmitted to our servers.

Settings: Stored locally to maintain your preferences across sessions.

Data Sharing

We do not have backend servers. All data remains on your device except screenshots sent directly to OpenAI.

Third-Party Service: OpenAI processes your screenshots according to their privacy policy: https://openai.com/privacy

We do not share data with any other third parties.

Your Rights

  • Add/remove your API key anytime via the extension popup
  • Delete all data by uninstalling the extension
  • Control features - enable/disable Auto-Solve, Ask mode, and UI visibility

Security

  • All data stored locally using Chrome's secure storage API
  • HTTPS encryption for OpenAI API communication
  • No server infrastructure to be compromised
  • Minimal permissions requested
  • Not intended for users under 13

Changes to Policy

Updates will be posted here with a new effective date. Significant changes will be communicated via the Chrome Web Store listing.
